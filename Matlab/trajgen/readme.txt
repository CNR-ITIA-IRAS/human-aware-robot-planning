1. Download the TypeII version of the Reflexxes Motion Library from http://www.reflexxes.com/products/overview-and-download.
2. Unzip the contents to this directory.
3. Compile the Simulink S-function using the script make_trajgen.m.
4. Open and run the Simulink model trajgen.slx.
5. If Simulink Coder is available, generate C++ code in Simulink by Code->C/C++ Code->Build Model. (Note that C code generation is not supported)
